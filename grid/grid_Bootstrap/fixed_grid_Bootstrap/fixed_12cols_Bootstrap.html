Bootstrap的12col fixed grid设置

Bootstrap默认的栅格系统为12列 ，形成一个940px宽的容器.

在12列、宽为940的固定宽度的grid系统中，供用到三个class，分别是：row用于定义行或者嵌套行；span用于定列宽；offset用于代表留空的列。具体示例如下：
